Versión español:
El archivo que contiene todo el código fuente es Space_time_github.ipynb, escrito en Python (con asistencia de IA). Este script carga el archivo puertos.xlsx, el cual contiene los datos correspondientes. Posteriormente, el script se ejecuta y genera automáticamente todas las carpetas y archivos necesarios para su publicación en Medium u otros medios.
